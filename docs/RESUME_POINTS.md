# Resume Points

## Project title

CineMatch AI Platform: Full-Stack Movie Recommendation and Authentication System

## Short description

Full-stack Python application with user authentication, movie discovery, watchlists, ratings, analytics, and content-based recommendations.

## Strong resume bullets

- Built a full-stack movie discovery platform using FastAPI, SQLite, HTML/CSS/JavaScript, and scikit-learn.
- Implemented secure user registration and login using password hashing, signed tokens, and protected API routes.
- Designed REST APIs for movie search, genre filtering, watchlist management, ratings, analytics, and recommendations.
- Developed a content-based recommendation engine using TF-IDF vectorization and cosine similarity over movie metadata.
- Added pytest coverage, Docker support, and GitHub Actions CI for a production-style project workflow.

## Skills shown

Python, FastAPI, REST APIs, SQLite, Authentication, scikit-learn, pandas, Machine Learning, JavaScript, Docker, Testing, GitHub Actions
